# OBS 音频设置指南 - AI Narrator 直播方案

## 音频流向原理

```
┌─────────────────────────────────────────────────────────────┐
│                        音频流向图                            │
└─────────────────────────────────────────────────────────────┘

TTS 语音 (系统声音)
       │
       ▼
┌──────────────┐      ┌──────────────┐      ┌──────────────┐
│   豆包客户端  │◄─────│  麦克风输入   │      │   OBS 直播   │
│  (听到提示)   │      │  (VB-Cable)  │      │  (观众听到)   │
└──────────────┘      └──────────────┘      └──────────────┘
       │                                            ▲
       │                                            │
       ▼                                            │
┌──────────────┐                          ┌──────────────┐
│  豆包语音输出 │─────────────────────────►│  桌面音频输入 │
│  (解说声音)   │                          │  (扬声器)    │
└──────────────┘                          └──────────────┘
                                                 ▲
                                                 │
┌──────────────┐                                 │
│   游戏声音    │─────────────────────────────────┘
│  (游戏音频)   │
└──────────────┘
```

## 核心原则

1. **观众听不到 TTS 提示音** - TTS 只传给豆包，不传给 OBS
2. **观众能听到豆包解说** - 豆包的声音传给 OBS
3. **观众能听到游戏声音** - 游戏声音传给 OBS

## 方案一：简单方案（推荐新手）

### 适用场景
- 不需要复杂的音频分离
- 使用物理双机或简单单机构型

### 设置步骤

#### 1. Windows 声音设置

1. 右键点击任务栏音量图标 → **声音设置**
2. **输出设备**：选择你的扬声器/耳机
3. **输入设备**：选择你的麦克风

#### 2. OBS 音频设置

1. 打开 OBS → **设置** → **音频**

2. **全局音频设备**：
   - 桌面音频：选择你的扬声器（如：扬声器/耳机）
   - 麦克风/辅助音频：选择你的麦克风

3. 点击 **确定** 保存

#### 3. 豆包客户端设置

1. 打开豆包桌面客户端
2. 发起语音通话
3. **麦克风选择**：选择你的物理麦克风
4. **扬声器选择**：选择你的扬声器

#### 4. 运行 AI Narrator

```bash
python doubao_narrator_simple.py
```

**音频流向**：
- TTS 声音 → 扬声器 → 豆包麦克风听到 → 豆包解说
- 豆包解说 → 扬声器 → OBS 桌面音频采集 → 直播
- 游戏声音 → 扬声器 → OBS 桌面音频采集 → 直播

**缺点**：观众可能隐约听到 TTS 声音（如果麦克风灵敏度高）

## 方案二：虚拟音频线路方案（推荐专业直播）

### 需要软件
- [VB-Cable](https://vb-audio.com/Cable/)（免费虚拟音频线）
- 或 [Voicemeeter](https://vb-audio.com/Voicemeeter/)（更专业的混音器）

### 安装 VB-Cable

1. 下载 VB-Cable：[https://vb-audio.com/Cable/](https://vb-audio.com/Cable/)
2. 解压，右键 **VBCABLE_Setup_x64.exe** → **以管理员身份运行**
3. 安装完成后重启电脑

### 设置步骤

#### 1. Windows 声音设置

1. 右键点击任务栏音量图标 → **声音设置**

2. **输出设备**：选择你的扬声器（正常播放声音）

3. **输入设备**：选择 **CABLE Output (VB-Audio Virtual Cable)**
   - 这样豆包会听到 TTS 声音

#### 2. OBS 音频设置

1. 打开 OBS → **设置** → **音频**

2. **全局音频设备**：
   - 桌面音频：选择你的扬声器
   - 麦克风/辅助音频：**禁用**

3. 在 OBS 主界面，**来源** 区域：
   - 添加 **音频输入采集**
   - 设备选择：**CABLE Output (VB-Audio Virtual Cable)**
   - 这样 OBS 不会采集到 TTS 声音

4. 点击 **确定** 保存

#### 3. 豆包客户端设置

1. 打开豆包桌面客户端
2. 发起语音通话
3. **麦克风选择**：**CABLE Output (VB-Audio Virtual Cable)**
   - 这样豆包只能听到 TTS，听不到其他声音
4. **扬声器选择**：你的扬声器

#### 4. 运行 AI Narrator

```bash
python doubao_narrator_simple.py
```

**音频流向**：
```
TTS 声音 ────────────────────────────────► VB-Cable ──► 豆包麦克风
                                               │
                                               ▼
                                            OBS 不采集

豆包解说 ──► 扬声器 ────────────────────────────────────► OBS 桌面音频
                                                            ▲
游戏声音 ──► 扬声器 ────────────────────────────────────────┘
```

**优点**：
- 观众完全听不到 TTS 提示音
- 音频分离清晰，直播质量高

## 方案三：Voicemeeter 高级方案（专业主播）

### 适用场景
- 需要精细控制多个音频源
- 需要实时混音
- 专业直播需求

### Voicemeeter 设置

1. 下载安装 Voicemeeter Banana
2. 设置虚拟输入：
   - Hardware Input 1: 你的麦克风
   - Hardware Input 2: VB-Cable (TTS)
   - Virtual Input: 桌面音频

3. 设置输出：
   - Hardware Out: 你的扬声器
   - Virtual Out: 给 OBS 的音频

4. OBS 设置：
   - 桌面音频：Voicemeeter VAIO
   - 麦克风：禁用

## 快速检查清单

在开播前，请确认：

- [ ] TTS 能正常播放声音
- [ ] 豆包能听到 TTS 声音（豆包有反应）
- [ ] 豆包能正常语音解说
- [ ] OBS 能采集到豆包的声音
- [ ] OBS 能采集到游戏声音
- [ ] **观众听不到 TTS 提示音**（重要！）

## 常见问题

### Q: 观众听到了 TTS 声音？
**A**: 
- 检查麦克风是否太灵敏，调低麦克风增益
- 使用方案二（VB-Cable）彻底分离音频
- 检查 OBS 音频来源设置

### Q: 豆包听不到 TTS？
**A**:
- 检查系统默认输入设备是否为 VB-Cable
- 检查豆包客户端麦克风设置
- 测试 TTS 音量是否足够大

### Q: OBS 采集不到豆包声音？
**A**:
- 检查 OBS 桌面音频设备设置
- 确认豆包使用扬声器输出，而不是耳机
- 检查 Windows 混音器，确认 OBS 有权限访问音频

### Q: 游戏声音和豆包声音不平衡？
**A**:
- 在 OBS 混音器中调整各音频源的音量
- 游戏声音建议 -10dB ~ -15dB
- 豆包声音建议 -5dB ~ -8dB

## 推荐设置参数

### OBS 音频滤镜（给豆包声音添加）

1. 在 OBS 混音器中，右键 **桌面音频** → **滤镜**

2. 添加 **压缩器**：
   - 比率：3:1
   - 阈值：-18dB
   - 攻击：6ms
   - 释放：60ms

3. 添加 **噪声抑制**：
   - 方法：NVIDIA 降噪（如果有 RTX 显卡）
   - 或 Speex 降噪（CPU 处理）

4. 添加 **增益**：
   - 增益：根据实际音量调整（通常 +5dB ~ +10dB）
