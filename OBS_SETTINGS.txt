========================================
OBS 正确设置指南
========================================

【问题】
耳机听到了 TTS、豆包有反应、OBS 也采集到了
→ 说明 OBS 采集了不该采集的音频源

【原因】
OBS 可能采集了以下之一：
- 桌面音频包含了 CABLE
- 麦克风/auxiliary 音频启用了
- 音频输入采集添加了 CABLE

【正确设置】

1. OBS 设置 → 音频 → 全局音频设备：

   桌面音频：你的扬声器（不是 CABLE！）
   麦克风/辅助音频：禁用

2. 来源区域检查：
   - 不要添加"音频输入采集"来源
   - 如果添加了，删除它或改为扬声器

3. 混音器检查：
   - 只应该有"桌面音频"
   - 不应该有"麦克风/Aux"

【音频流向】

TTS → CABLE Input → CABLE Output → 豆包（听到）
                          ↓
                       监听 → 耳机（你听到）

游戏声音 → 扬声器 → OBS 桌面音频 → 直播（观众听到）
豆包解说 → 扬声器 → OBS 桌面音频 → 直播（观众听到）

【关键】
- TTS 走 CABLE 线路
- OBS 只采集扬声器
- 两者完全分离！

【检查清单】
□ OBS 设置 → 音频 → 桌面音频 = 扬声器
□ OBS 设置 → 音频 → 麦克风/Aux = 禁用
□ OBS 来源 → 没有"音频输入采集"
□ 混音器 → 只有"桌面音频"

设置完成后：
- 观众能听到游戏声音 ✓
- 观众能听到豆包解说 ✓
- 观众听不到 TTS ✓
